解读《工业物联网白皮书（2017版）》_背景和内涵

发表于 2018-01-20 | 分类于 物联网

在2017年9月，中国电子技术标准化研究院发布了最新的《工业物联网白皮书》。从该白皮书中我们可以窥视到当前国内业界对工业互联网的定义，现状的理解以及未来发展的展望。

发展背景

发展现状：工业物联网推动产业转型升级
工业物联网应用的出现，主要基于两个背景现状，一个是物联网技术本身的发展，再一个就是现在各国都提出的智能制造的战略方向（有点像三国鼎立，中国制造2025，美国先进制造伙伴计划，德国工业4.0）。这两个背景叠加在一起就形成了”物联网 + 制造业” = 工业物联网。其应用于生产制造的各个生命周期环节(如下图)。这种智能变革带来的好处是使得传统的产业得到改造，以生产驱动的经济发展方式向创新驱动的方式转变。从而促进整个产业结构的升级。

发展需求：工业物联网契合产业应用需求
目前可以看到工业领域，已经有很多基于工业物联网的应用来解决实际问题。大概可以分为以下这么几种应用：

  • 实时监控。 这里面包括对设备状态，设备所处环境状态，原材料，在制品和人员的状态。全面掌握生产过程的各种数据，才能做到制造过程的智能执行。
  • 物料管理与生产过程的连接。 通过RFID技术，对仓库中库存物料存储情况做到精准感知。在连接生产系统后，可以利用物料的仓库情况来进行更细粒度的生产排产。
  • 设备的预防性诊断。 相比以前的定期诊断和维护，预防性诊断能够更及时的预知可能发生的设备故障，从而保障生产过程不会因为突发故障而造成停产损失。或者减少有些可用部件因定期维护更换所造成的不必要的损失。

发展意义： 工业物联网主力智能制造
分三个层面来看。从国家层面上来将，意义在于产业结构的升级调整。从以前粗放式生产，变为精细化，节约型的生产方式。从企业层面来讲，工业物联网的技术应用于制造活动的各个环节，使得生产的效率更高，能耗等到节省。从个人层面来说，因为系统的自动化程度更高，更智能，使得人更能从繁重的工作中脱离出来。（但是也会带来失业的风险）

内涵

工业物联网从本质上讲要大于简单的 “工业 + 物联网”。其原因是工业领域自身原有的一些软硬件系统的，比如控制系统，工业软件。物联网技术要和这些原有系统相互结合，并且还要面临工业领域中更高的安全性，更恶劣的现场环境等问题。

从下图中我们可以看到《白皮书》对工业物联网的本质的描述。

《白皮书》给出的工业物联网的定义为：
工业物联网是通过工业资源的网络互联、数据互通和系统互操作，实现制造原料的灵活配置、制造过程的按需执行、制造工艺的合理优化和制造环境的快速适应，达到资源的高效利用，从而构建服务驱动型的新工业生态体系。

从这个定义可以看出互联互通是方法，达到生产过程中各个环节的高效是目标。

白皮书定义的工业物联网的六大典型特征：

  • 智能感知
  • 泛在连通
  • 精准控制
  • 数字建模
  • 实时分析
  • 迭代优化

工业物联网的阶段包括如下：

  • 智能的感知控制
  • 全面的互联互通
  • 深度的数据应用
  • 创新的服务模式

阶段的认识对于帮助企业明确工业4.0的实时步骤是很有好处的。在没有前面阶段的实施的情况下，跳到其他阶段是不具有可行性的。

在向客户提供工业物联网的技术服务时，最需要show出来，去打动客户的部分就是你给客户带了什么价值。白皮书给出的价值定义如下：

  • 提升价值（准确的是说提升数据的价值）
  • 优化资源（工业资源得到最大利用）
  • 升级服务（将数据变为资产）
  • 激发创新（资源高效利用，营造创新环境）

关于工业物联网的架构会在之后的文章中说明。
